Setup

Putting on the Strap

The strap buckle should be facing out when strapped onto the bass.

Squeeze the top strap post through the strap's leather end. The leather will likely be pretty tight so put some force into it.

Make sure the strap isn't twisted and put the other leather end on the strap post.

Amp and Guitar Plugin

Plug one end of the 1/4" cable into the guitar's output. Plug the other end into the amp input. The amp might have separate inputs for passive and active basses. Your bass is active if it takes batteries.

Plug the power cable into the back of the amp and the plug into the outlet.

Flip the power switch and make sure everything sounds ok. If there is no sound when you pluck a string or the amp just makes loud static noises, unplug everything and repeat the process. Refer to troubleshooting if the problem persists.

Tuning

Recommended Tuning Method for First Time

If the bass is extremely out of tune, it is possible to tune the strings to the right note but an entire octave (series of notes ABCDEFG) too low or too high. The video provides an absolute reference for each string.

Tune from the thickest string to the thinnest string, following along with the video.

Play the indicated string with the video and tune the note so that they sound identical. Turn the tuning key towards you to make the note lower and away from you to make the note higher.

Repeat for each string.

Tuning with a Clip On Tuner or Digital Tuner

If using a clip-on tuner, clip the tuner onto the headstock. Alternatively one may use an online tuner (like this one) which will function almost the same way. Make sure the tuner is set to the correct instrument - look for a B on the tuner.

Too high

Too low

Work from the thickest string up to the thinnest string. The strings should be from thickest to thinnest, E-A-D-G. Pluck the thickest string and you should see the tuner display a note with an indicator for if the bass is too low or too high.

Turn the tuning keys to adjust the note until the tuner indicates the string is in tune. Turn the key towards you to make the note higher, away from you to make the note lower.
